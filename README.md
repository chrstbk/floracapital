# Flora Capital Partners Website

Official website for Flora Capital Partners - Long-term business owners committed to growing established B2B service companies in Ontario.

## Built With

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

## Development

```bash
npm install
npm run dev
```

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.
